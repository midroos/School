# نظام إدارة مالية للمدارس

هذا المشروع هو نظام إدارة مالية بسيط للمدارس، تم تطويره باستخدام Flet. يوفر واجهة مستخدم رسومية لإدارة الطلاب، وإنشاء خطط الرسوم، وتتبع المدفوعات.

## الميزات

*   **لوحة التحكم بالتحصيل:** عرض الإيرادات اليومية، وعدد الأقساط المتأخرة، وسجل بالمعاملات الأخيرة.
*   **إدارة الطلاب:** إضافة وتعديل وعرض بيانات الطلاب.
*   **الخطط المالية:** إنشاء خطط أقساط للطلاب.
*   **تسجيل المدفوعات:** تسجيل المدفوعات النقدية والتحويلات البنكية والشيكات.

## كيفية التشغيل

1.  **تثبيت المتطلبات:**
    ```bash
    pip install -r school/requirements.txt
    ```
2.  **تشغيل التطبيق:**
    ```bash
    python school/main.py
    ```
